# Avatar 形象生成 Prompt

配置文件：`src/pipeline/avatar-style.prompt.ts`

## System Prompt

指导 LLM 根据文章内容为主播设计"亲历者"形象。

**核心约束**：
- 手机前置摄像头拍摄风格（轻微过曝、自然噪点、超写实）
- 人物情绪饱满，不要面无表情
- 人物正面面对镜头，半身照
- 根据文章内容设计身份和妆造

**示例映射**：
- 外卖大战 → 外卖骑手
- 企业财报 → 交易所交易员
- 波斯湾战争 → 战地记者
- 美食文化 → 厨师

## 图片模型配置

| 参数 | 值 | 说明 |
|------|-----|------|
| `model` | Nano Banana 2 | 免费模型 |
| `aspectRatio` | 3:4 | 竖版半身照 |
| `resolution` | 1K | 分辨率 |
| `timeout` | 600 | 超时秒数 |
